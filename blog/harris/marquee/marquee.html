Marquee

Marquee adalah salah satu fungsi untuk memodifikasi tulisan pada website/blog. Marquee ini untuk membuat efek teks menjadi bergerak atau berjalan.

Dalam tag Marquee ini mempunyai beberapa attribute seperti dibawah ini :

- bgcolor : untuk mengatur background dari text marquee
- direction : untuk mengatur gerakan text (up,down,left,right)
- scrollamount : untuk mengatur kecepatan gerakan text
- behaviour : untuk mengatur effect dari gerakan
- width : untuk mengatur lebar area gerakan
- height : untuk mengatur tinggi area gerakan
- align : untuk mengatur posisi
- onmouseover : untuk menghentikan effect ketika mouse mendekat
- onmouseout : untuk menjalankan effect ketika mouse menjauh

Jenis-Jenis Marquee

1. Bergerak dari kanan ke kiri

syntax:


             <marquee><h3>Asbari.com </h3></marquee>

Output:

Asbari.com

2. Bergerak dari kiri ke kanan

syntax:


         <marquee direction="right"><h3>Asbari.com </h3></marquee>

Output:

Asbari.com

3.Bergerak bolak-balik dan memantul

syntax :


        <marquee behavior="alternate"><h3>Asbari.com </h3></marquee>

Output:

Asbari.com

4. Bergerak dari atas ke bawah

syntax :


        <marquee direction="down"><h3>Asbari.com </h3></marquee>

Output:

Asbari.com

5. Bergerak dari bawah ke atas

syntax :


            <marquee direction="up"><h3>Asbari.com </h3></marquee>

Output:

Asbari.com

6. Bergerak menyudut dari bawah ke atas

syntax :


    <marquee><marquee direction="up"><h3>Asbari.com </h3></marquee></marquee>

Output:

Asbari.com

7. Bergerak menyudut dari atas ke bawah

syntax :


    <marquee><marquee direction="down"><h3>Asbari.com </h3></marquee></marquee>

Output:

Asbari.com

8. Bergerak dengan kecepatan tertentu

syntax:


        <marquee behavior="scroll" direction="left" scrollamount="5">Asbari.com (Slow speed)</marquee>
        <marquee behavior="scroll" direction="left" scrollamount="10"> Asbari.com (Medium speed)</marquee>
        <marquee behavior="scroll" direction="left" scrollamount="17"> <Asbari.com (Fast speed);/marquee>

Output:

Asbari.com (Slow speed) Asbari.com (Medium speed) Asbari.com (Fast speed)
