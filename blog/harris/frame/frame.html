FRAME

Frame adalah teknik yang digunakan untuk memanggil beberapa halaman kemudian ditampilkan dalam satu halaman.

Format Penulisan tag Frame :


        <frameset>... frameset>

Format penulisan elemen tag frame :


        <frame>... </frame>

Tag Frameset memiliki atribut diantaranya :

- Border : menentukan garis border
- Cols : menentukan lebar kolom (px/%/*),dan untuk menentukan jumlah kolom yaitu memakai tanda koma(,)
- Rows : menentukan panjang baris (px/%/*), dan untuk menentukan jumlah baris yaitu memakai tanda koma (,)

Tag Frame memiliki atribut diantaranya :

- Name : untuk menentukan nama frame atau untuk target link
- Scr : berfungsiuntuk menunjukkan/lokasi sumber file

Buat File baru dengan nama index.html kemudian simpan dalam folder frame


        <!DOCTYPE html>
        <html>
        <Head>
        <Title>Belajar HTML : FRAME </Title>
        </head>
        <frameset rows"20%,*10%" border="1">
        <frameset name="header" src="header.html"></frame>
        <frameset cols="25%,*" border="1">
        <frameset name="sidebar" src="sidebar.html"></frame>
        <frameset name="konten" src="konten.html"></frame>
        </frameset>
        <frameset name="footer" src="footer.html"></frame>
        </frameset>
        </html>
