FRAME

  • Elemen dalam HTML digunakan untuk membangun web page yang memungkinkan penampilan beberapa web page ditampilkan dalam satu window browser. Window dalam browser dibagi menjadi beberapa windows yang disebut frame. Model ini sering digunakan untuk lebih memperjelas penyajian informasi, dengan menu sajian selalu ditampilkan. Fasilitas ini hanya dapat dijalankan pada browser web Internet Explorer 3.0 ke atas dan Netscape Navigator 2.0 ke atas.

Contoh Frame

Frame 1

  • Buatlah frame pertama dengan nama halaman awal.html

    <html>
    <body bgcolor="grey">
        <center>
            <h1>Halaman ini dengan menggunakan</h1>
            <h1>Frame</h1>
            <p><strong>Selamat Datang</strong></p>
            <p><strong>Email: <a href="Juli.com">Juli.com</a></strong></p>
        </center>
    </body>
</html>

Frame 2

  • Buatlah frame kedua dengan nama halaman menu.html

    <html>
    <head>
        <title>Membuat Marquee pada status bar</title>
    </head>
    <body bgcolor="lavender">
        <center>
            <p> <font color="orange" size="5">Latihan Marquee Javascript</font></p>
            <body onload="p()">
                <marquee direction="right"> Teks ke kanan</marquee>
                <marquee direction="left"> Teks ke kiri</marquee>
    </body>
</html>

Frame 3

  • Buatlah frame ketiga dengan nama contoh_frame.html

    <html>
    <head>
        <title>Latihan Frame</title>
    </head>
    <frameset cols="30%,*">
        <frame src="halaman awal.html" name="fr1"></frame>
        <frame src="halaman menu.html" name="fr2"></frame>
    </frameset>
</html>
