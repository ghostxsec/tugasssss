MARQUEE

Marquee adalah salah satu kode bahasa program HTML, untuk membuat efek tulisan / gambar menjadi berjalan atau bergerak.

Tag marquee memiliki atribut sebagai berikut:

Atribut value Keterangan
behavior scroll,slide,alternate Mendefinisikan jenis scrolling
direction up,down,left,right Mendefinisikan arah scrolling
height pixel atau % Menyatakan tinggi marquee

Sebagai catatan marquee memiliki 3 sifat atau behaviour:
- Scroll : marquee menghilang ketika menabrak sisi kiri (kode default)
- Slide : marquee berhenti ketika menabrak sisi kiri
- Alternate : marquee memantul ketika menabrak sisi

jenis-jenis Marquee:

1. Bergerak ke kiri:

syntax:


             <marquee><h3>Kelompok 2 </h3></marquee>

Output:

Kelompok 2

2. Bergerak ke kanan:

syntax:


        <marquee direction="right"><h3>Kelompok 2 </h3></marquee>

Output:

Kelompok 2

3. Bergerak ke atas:

syntax:


        <marquee direction="up"><h3>Kelompok 2 </h3></marquee>

Output:

Kelompok 2

4. Marquee dengan attribute behaviour alternate:

syntax:


        <marquee behavior="alternate"><h3>Kelompok 2 </h3></marquee>

Output:

Kelompok 2

5. Bergerak dengan kecepatan tertentu

syntax:


        <marquee behavior="scroll" direction="left" scrollamount="3"> Slow speed</marquee>
        <marquee behavior="scroll" direction="left" scrollamount="7"> Medium speed</marquee>
        <marquee behavior="scroll" direction="left" scrollamount="15"> Fast speed</marquee>

Output:

Slow speed Medium speed Fast speed
