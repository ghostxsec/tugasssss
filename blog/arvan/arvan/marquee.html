Marquee

Elemen Marquee digunakan untuk membuat teks atau media lain (seperti gambar atau video) bergerak baik secara horizontal maupun vertikal. Pada Marquee terdapat beberapa atribut seperti width,height,up and down

BERIKUT CONTOH SIMPEL KODE PENULISAN WIDTH DAN HEIGHT

1. Atribut Width dan Height

syntax:


            <marquee><h3>Teks ini berjalan menggunakan atribut width dan height </h3></marquee>

Output:

Atribut Width dan Height di Tag Marquee Teks ini berjalan menggunakan atribut width dan height

BERIKUT CONTOH SIMPEL KODE PENULISAN ATRIBUT DIRECTION

2. Atribut Direction LEFT dan RIGHT

syntax:


          <marquee direction="right"><h3>Teks ini Berjalan menggunakan atribut right.com </h3></marquee>
          <marquee direction="left"><h3>Teks ini Berjalan menggunakan atribut left.com </h3></marquee>

Output:

Atribut Direction di Tag Marquee Teks ini ke kanan Teks ini ke kiri

3. Atribut Direction UP dan DOWN

syntax:


        <marquee direction="up"><h3>Teks ini Berjalan menggunakan atribut up.com </h3></marquee>
        <marquee direction="down"><h3>Teks ini Berjalan menggunakan atribut left.com </h3></marquee>

Output:

Atribut Direction di Tag Marquee Teks ini bergerak dari bawah ke atas
Teks ini bergerak dari atas ke bawah

BERIKUT CONTOH SIMPEL PENULISAN KODE ATRIBUT BGCOLOR

4. Atribut bgcolor

syntax:


        <marquee><marquee bgcolor="black""up"><h3>Teks ini berjalan menggunakan atribut bgcolor.com </h3></marquee></marquee>

Output:

Atribut BGColor di Tag Marquee Teks ini berjalan menggunakan atribut bgcolor
