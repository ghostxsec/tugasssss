A. [CTF KKS-TNI AD 2021] Write Up Forensic Another User

26-06-2022

Diberikan sebuah file bernama vm_mahasiswa.zip sebesar 2GB yang ketika diekstrak berisi sebuah virtual machine bernama Miss_us_final.ova yang disita dari terduga pelaku peretasan

Kemudian kita bisa membukanya menggunakan dengan virtual box. Seperti di soal kita memiliki akses dengan menggunakan credential guest:guest. Di dalam kasus ini kita disuruh mencari akun selain akun guest.

lalu langsung mencari beberapa informasi user pada sistem dengan melihat file /etc/passwd.

Dan kita akan mendapatkan beberapa nama akun root,ellen dan guest. Kemudian kita bisa mencoba melihat informasi tentang kata sandi pengguna pada file /etc/shadow.

Terdapat 2 akun yang memiliki password terenkripsi SHA-512. Karena kita sudah login menggunakan akun guest. Berarti kita harus mencari password dari akun ellen. Ini sangat mudah. Saya langsung berpikir untuk melakukan brute force SSH server untuk mencari password dari akun ellen dengan menggunakan modul auxiliary/scanner/ssh/ssh_login yang terdapat di metasploit

Bingo!!! Kita mendapatkan password dari akun ellen yaitu ihateyou Ini biasa disebut dengan weak password. Cukup dengan menggunakan wordlist yang tersebar di google kita bisa mendapatkannya. Lalu kita mencoba test login dengan credential yang kita dapat tadi dan itu berhasil.

Flag : KKST2021{ellen:ihateyou}
